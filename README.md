# Netflix Landing Page Clone
A Modern and Responsive UI built using HTML and CSS
This project is a fully responsive Netflix Landing Page Clone developed using pure HTML and CSS.
It replicates the UI and layout of Netflix’s homepage, focusing on clean structure, responsive design,
and modern styling. 

## Features
- Responsive navigation bar
- Full-screen hero section with background image
- Styled headings and call-to-action buttons
- Multiple feature sections (text + images)
- FAQ section (accordion style – optional)
- Clean and minimal footer
- Fully responsive for mobile, tablet, and desktop

## Technologies Used
- HTML5
- CSS3 
- Media Queries for responsive design
