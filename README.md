# Netflix Landing Page Clone

A modern and responsive Netflix landing page clone built using HTML and CSS.  
The project recreates the layout and visual design of the Netflix homepage, focusing on clean structure, responsive design, and modern UI styling.

---

## Project Description

This project replicates the user interface of the Netflix landing page using only HTML and CSS. It demonstrates how modern web layouts can be built with structured HTML and styled using CSS while maintaining responsiveness across different devices.

The project focuses on layout design, responsive styling, and creating visually appealing UI sections similar to the Netflix homepage.

---

## Features

- Responsive navigation bar
- Full-screen hero section with background image
- Styled headings and call-to-action buttons
- Feature sections with text and images
- FAQ section layout
- Clean and minimal footer
- Fully responsive design for mobile, tablet, and desktop

---

## Technologies Used

- HTML – Page structure  
- CSS – Styling and layout  
- Media Queries – Responsive design  

---

## Learning Outcomes

- Building responsive layouts using CSS
- Working with flexbox and modern layout techniques
- Structuring web pages with semantic HTML
- Implementing responsive design using media queries

## Running the Project Locally

1. Clone the repository

```
git clone https://github.com/rushikesh-auti/Netflix-Landing-Page-Clone.git
```

2. Navigate to the project directory

```
cd Netflix-Landing-Page-Clone
```

3. Open `index.html` in your browser
